# LLMs are the primary daily tool in the stack

## Statement

LLMs are a great tool. I use them daily more than any other tool in my stack.

That is the whole claim. No cognitive substrate mysticism, no "extension of self." A factual daily-use observation: LLMs top the list of tools by frequency of use, across code, writing, data, learning, and collaboration.

This is the operational complement to `belief.ai-pm-skillset-table-stakes`. Table-stakes is the positional claim ("AI fluency is now a non-negotiable for PMs"). This belief is the operational reality: here is what daily use actually looks like.

## Origin

The first evidence is a March 2023 post where ChatGPT-3 was asked to write its own testimonial about working with Agam, then published as the post body. A move that telegraphed the belief before it got named. Two weeks later (2023-03-14), the manifesto post listed six concrete daily surfaces: data analysis with Jupyter and Pandas and SQL, PRD refinement, brainstorming, Linux home-lab setup, home network security, note-structuring. The note-structuring line is the seed that eventually becomes the second-brain: "My notes have never been so actionable thanks to ChatGPT's help."

By May 2023, "my LLM friend" entered the vocabulary with no scare-quotes. The framing had stabilized within two months of the March anchor.

## Refinement arc

The tools in the stack changed. The daily-use fact did not.

2024-Q1-Q3: the stack hardened into a multi-substrate configuration. Open-source LLMs deployed locally, ChatGPT Pro, GitHub Copilot, Bing AI with integrated browsing, plus several custom GPT wrappers built for specific personal use cases: a data analytics helper, a writing helper, a PRD summarizer. The stack was configured around personal workflows, not generic use cases.

2024-06-05: "ChatGPT was down yesterday... this prompted me to be prepared for the times when I have to be off-the-grid but with my trusty LLM peer." When the primary goes down, download ollama and bind LLAMA3 and PHI3 to a keyboard shortcut. Continuous LLM access is non-negotiable; the specific substrate is interchangeable.

2024-05-31: "I have created a personal GPT wrapper for each subject that I am after. I tend to look up top programs, look at their curriculum and add that as a structure to my wrappers. This has replaced my podcast addiction with a more interactive version." One concrete, specific displacement of a prior habit. Not a feature; a shift in how learning works day to day.

2024-07-19: "It's AI that is helping me code. AI building AI is now." Recursion: LLM extends the person, person extends the LLM into the next build.

2025-2026: Claude Code as the primary coding and build partner. The second-brain project itself (2026-04-21/23) is the current live artifact. "Funnily this has become my preferred way to use claude." The second-brain is downstream of this belief: it exists because the daily-use volume made a persistent context layer worth building.

Note in the Refinement arc: the old slug was `belief.llm-as-voice-extension`. That name was misleading on two counts: "voice" suggested voice-AI products (separate domain, see `voice-ai-craft`), and "extension" implied a cognitive-substrate framing the actual belief does not carry. Slug renamed and statement reframed at ontology lock 2026-04-26.
