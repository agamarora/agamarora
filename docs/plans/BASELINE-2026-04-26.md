## Architecture

```
scripts/
├── lib/
│   └── chrome.mjs          single source of truth — exports:
│                             SHARED_PRELOAD_HTML
│                             SHARED_FONTFACE_CSS
│                             SHARED_TOKENS_CSS
│                             SHARED_RESET_CSS
│                             SHARED_ICON_CSS
│                             SHARED_HEADER_CSS
│                             SHARED_AAMARK_CSS
│                             SHARED_FOCUS_CSS
│                             SHARED_CHROME_CSS  (bundled)
│                             SVG_SPRITE
│                             SHARED_HEADER_HTML
│                             SHARED_AAMARK_HTML
│                             AAMARK_SCRIPT
│
├── build-wiki.mjs          imports chrome.mjs; regenerates 36 wiki pages
├── build-kg.mjs            generates wiki/kg.json from ontology
└── sync-chrome.mjs         rewrites marker-bounded chrome blocks in 10 hand pages
```

**To change the header / sprite / aa-mark / chrome CSS site-wide:**

```bash
edit scripts/lib/chrome.mjs
npm run sync:chrome      # rewrites hand pages
npm run build            # regenerates wiki pages
git add . && git commit && git push
```

That's it. One edit, two commands. No more 9-file sweeps.
