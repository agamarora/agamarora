## About this repository

This repo **is** [agamarora.com](https://agamarora.com) — the source code, content, design, and serverless function behind the live site. It is published openly so you can read how it is built. It is **not** open-source for reuse.

**Stack.** Vanilla HTML/CSS/JS, no framework, no build step (except a small KG/wiki generator). One Netlify serverless function that streams responses from Groq for the live AI agent at `/enter`. Self-hosted Satoshi variable font. Inline SVG sprite for icons (zero external CSS).

**Why open?** I learn faster from real codebases than tutorials. If reading mine helps you ship something of your own, take the ideas — not the code.
